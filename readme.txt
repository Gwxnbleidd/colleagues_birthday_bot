ТГ бот для оповещение о днях рождения коллег.

в main лежит бот и start_polling
в handlers лежат все обработчики телеграмм бота
в keyboards лежат клавиатуры для тг бота

в models лежат функции работы с базой данных.
Как БД я выбрал PostgreSQL, у меня две таблички users с полями id,name,birthday и subscriptions c полями user_id (id пользователя) и subscription_id (id того, на кого подписан пользователь) 

в endpoints лежит fastapi и фоновая задача для проверки дней рождений сегодня.
в shemas - класс User из pydantic

в config лежат все для подключения к БД, TOKEN ТГ бота и URL fastapi, его получил с помощью tuna


Для запуска бота надо запустить main.py и endpoints.py (я хотел их объединить в одном файле, но возникло много ошибок, и я побоялся, что не успею уложиться в дедлайн по заданию, поэтому оставил так)
