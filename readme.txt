ТГ бот для оповещения о дне рождения коллег.

Нужно подписаться на пользователя, тогда вам придет сообщение в день его рождения.

Прочитал в интернете, что питоновские библиотеки не могут создать чат, а могут лишь создать ссылку на чат и отправить ее пользователям.
Тогда функцию приглашения в группу с днем рождения можно реализовать так: Сделать администратора, у которого будет свой роутер, 
в который включен роутер обычного пользователя, и у него будет функция отправки ссылки на чат. Тогда администратор должен самостоятельно
создавать группу, а ссылку и приглашение бот разошлет автоматически.
Либо можно работать с telegramAPI, можно получить API ключ и с помощью метода messages.createChat создать чат, там же можно пригласить пользователей.
